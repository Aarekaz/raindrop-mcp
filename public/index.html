Prompts that use the web you already saved.

Raindrop MCP connects Raindrop.io to OAuth-aware MCP clients, so your LLM can search bookmarks, inspect collections, read tags, and work with highlights through real MCP tools.

See prompt examples
bookmark_search Find saved pages by text, tags, collection, type, date, or favorite status.
bookmark_manage Create, update, delete, or get save suggestions for a URL.
tag_list Show the vocabulary in your saved library or a specific collection.
highlight_manage List, create, update, or delete highlights attached to saved pages.
Prompt lab raindrop-mcp /mcp
Find the links I saved about MCP OAuth and tell me which one covers protected-resource metadata.
bookmark_search Search all bookmarks for "MCP OAuth protected resource" with a small result count.
resources/read Open the selected raindrop://bookmark/{id} resource if the client wants the full saved record. 
{
  "tool": "bookmark_search",
  "arguments": {
    "search": "MCP OAuth protected resource",
    "collection": 0,
    "perPage": 5
  },
  "returns": "Saved links, titles, excerpts, tags, and raindrop://bookmark/{id} resources"
}

Concrete prompts, real MCP tool calls.

Each example maps a normal user request to the tool surface this server exposes. IDs shown below stand in for values returned by earlier tool calls.

Find a saved article by memory

"I saved something about OAuth resource metadata. Find the official spec and summarize why it matters for MCP clients."

bookmark_search { "search": "OAuth Protected Resource Metadata", "collection": 0, "perPage": 5 }

Save a repo while working

"Save the MCP TypeScript SDK repo, tag it mcp and sdk, and mark it important so I can find it later."

bookmark_manage { "operation": "create", "url": "https://github.com/modelcontextprotocol/typescript-sdk", "tags": ["mcp", "sdk"], "important": true }

Review research tags

"Show me my top tags, then suggest which AI-agent tags look redundant before changing anything."

tag_list { "collectionId": 0 }

Pull highlights from a source

"Find my saved Anthropic article and pull the highlights I left on it." The second call uses the bookmark id returned by search.

bookmark_search { "search": "Anthropic", "collection": 0, "perPage": 3 } highlight_manage { "operation": "list", "bookmarkId": 912345678 }

Find unsorted cleanup work

"Find my untagged saved links about Cloudflare Workers so I can clean up the collection."

bookmark_search { "search": "Cloudflare Workers #notag", "collection": 0, "perPage": 10 }

Use collections as context

"List my collections, then search the relevant one for posts about direct HTTP MCP transport."

collection_list {} bookmark_search { "search": "direct HTTP MCP transport", "collection": 0, "perPage": 5 }

Add Raindrop MCP to your client

Use the MCP endpoint. Your client discovers OAuth, opens Raindrop authorization, and stores the resulting token.

Add to Claude Desktop MCP config.

{
  "mcpServers": {
    "raindrop": {
      "url": "https://raindrop-mcp.anuragd.me/mcp",
      "transport": "streamable-http"
    }
  }
}

Add to Cursor MCP config.

{
  "mcpServers": {
    "raindrop": {
      "url": "https://raindrop-mcp.anuragd.me/mcp",
      "transport": "streamable-http"
    }
  }
}

Register the hosted MCP endpoint.

codex mcp add raindrop https://raindrop-mcp.anuragd.me/mcp

Use this with any OAuth-aware Streamable HTTP MCP client.

{
  "name": "raindrop",
  "url": "https://raindrop-mcp.anuragd.me/mcp",
  "transport": "streamable-http"
}
