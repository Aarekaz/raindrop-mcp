What it reaches

The public API surface uses the same names your MCP client sees.

Bookmarks

Search saved links, create new entries, move items, and fetch bookmark detail when the client needs context.

bookmark_search
Collections

List top-level and nested collections so your client can work with the structure you already maintain.

collection_list
Tags

Explore and manage the tags that make your library useful across projects, research, and reading queues.

tag_list
Highlights

Read or write highlighted passages and notes in context, attached to the original saved item.

highlight_manage

Authorization stays explicit

Clients discover the protected resource metadata, start OAuth, and receive scoped tokens. Bookmark content stays in Raindrop and is fetched live when a tool runs.

{
  "mcpServers": {
    "raindrop": {
      "url": "https://raindrop-mcp.anuragd.me/mcp",
      "transport": "streamable-http"
    }
  }
}

Add Raindrop MCP to your client

Use the MCP endpoint. Your client discovers OAuth, opens Raindrop authorization, and stores the resulting token.

Add to Claude Desktop MCP config.

{
  "mcpServers": {
    "raindrop": {
      "url": "https://raindrop-mcp.anuragd.me/mcp",
      "transport": "streamable-http"
    }
  }
}

Add to Cursor MCP config.

{
  "mcpServers": {
    "raindrop": {
      "url": "https://raindrop-mcp.anuragd.me/mcp",
      "transport": "streamable-http"
    }
  }
}

Register the hosted MCP endpoint.

codex mcp add raindrop https://raindrop-mcp.anuragd.me/mcp

Use this with any OAuth-aware Streamable HTTP MCP client.

{
  "name": "raindrop",
  "url": "https://raindrop-mcp.anuragd.me/mcp",
  "transport": "streamable-http"
}
