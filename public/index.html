Add Raindrop MCP to your client

Use the MCP endpoint. Your client discovers OAuth, opens Raindrop authorization, and stores the resulting token.

Add to Claude Desktop MCP config.

{
  "mcpServers": {
    "raindrop": {
      "url": "https://raindrop-mcp.anuragd.me/mcp",
      "transport": "streamable-http"
    }
  }
}

Add to Cursor MCP config.

{
  "mcpServers": {
    "raindrop": {
      "url": "https://raindrop-mcp.anuragd.me/mcp",
      "transport": "streamable-http"
    }
  }
}

Register the hosted MCP endpoint.

codex mcp add raindrop https://raindrop-mcp.anuragd.me/mcp

Use this with any OAuth-aware Streamable HTTP MCP client.

{
  "name": "raindrop",
  "url": "https://raindrop-mcp.anuragd.me/mcp",
  "transport": "streamable-http"
}
